# Personal Financial Dashboard

Personal Finance Dashboard is a modern financial management system designed to help users efficiently manage their finances, track expenses, and plan for the future in one centralized platform. The system provides real-time insights into financial health, spending patterns, and investment opportunities, allowing users to monitor and control their financial operations smoothly.

Personal Financial Dashboard is built with a modern full-stack architecture using Next.js for the frontend and Strapi as a headless CMS backend.

- [![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)

### Features

- Transaction Management
  Add, update, and monitor all trucks in the financial.
- Dashboard Analytics
  Visual charts to display load distribution and financial availability.
- Authentication System
  Secure login and protected dashboard routes.
- Modern UI
  Clean and responsive interface for a smooth user experience.
